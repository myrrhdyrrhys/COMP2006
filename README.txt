--------------------------
        Scheduler
--------------------------

Purpose:
    A program to model running two methods of CPU scheduling, Priority Pre-emptive and Shortest Remaining Time First, concurrently.

Compiling:
    Program uses the provided makefile to compile.
    Simply enter "make" (not including the quotations) when in the main directory compile the program.

Execution:
    In order to run the Scheduling simulator, enter:
        "./Scheduler"   (not including the quotations)
    The program should provide a prompt "Scheduling Simulation: " to which you should enter the filename of the source file containing the list of processes to be processed (e.g. "simInput").
    If instead you wish to quit the simulator at any time, enter "QUIT" (not including quotations) when the prompt is provided.
